# ✨ Text Summariser App

This project is a simple and powerful AI-based text summarisation web application built using Streamlit and Google Gemini API.  
It allows users to paste any large text and instantly generate a concise and meaningful summary.  
The app ensures clarity by focusing on key points and presenting them in easy-to-understand language.  
It also includes features like word count display and a clean, modern UI for better user experience.  
This prototype demonstrates how generative AI can be used to simplify information and improve productivity.

## 🚀 Features
- AI-powered summarisation using Gemini
- Clean and modern UI
- Word count tracking
- Error handling for API and input validation

## 🛠️ Tech Stack
- Python
- Streamlit
- Google Generative AI (Gemini)
